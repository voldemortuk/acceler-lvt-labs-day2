# MCP Chat

MCP Chat is a command-line application for interactive chat with Claude (the
Anthropic API), extended with the **Model Context Protocol (MCP)**. It supports
document retrieval, command-based prompts, and pluggable tool integrations via
MCP servers.

This is a TypeScript / Node.js application.

## Prerequisites

- Node.js 20 or newer
- An Anthropic API key ([console.anthropic.com](https://console.anthropic.com/))

## Setup

### 1. Configure environment variables

Copy `.env.example` to `.env` and fill in the values:

```bash
cp .env.example .env
```

```ini
# .env
CLAUDE_MODEL="claude-sonnet-4-5"   # the Claude model to use
ANTHROPIC_API_KEY=""               # your Anthropic API secret key
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run the project

For development (runs directly from TypeScript source via `tsx`):

```bash
npm run dev
```

For a production-style run (compile to `dist/`, then run the compiled output):

```bash
npm run build
npm start
```

## Usage

### Basic interaction

Type your message and press Enter to chat with the model:

```
> What is the condenser tower report about?
```

### Document retrieval

Use the `@` symbol followed by a document ID to include that document's content
in your query:

```
> Tell me about @deposition.md
```

Press **Tab** after typing `@` (and a partial name) to autocomplete document IDs.

### Commands

Use the `/` prefix to run a prompt defined by the MCP server:

```
> /format deposition.md
```

Press **Tab** to autocomplete command names, and Tab again after the command to
autocomplete the document ID argument.

> **Completion is Tab-triggered.** Unlike a live pop-up menu, completions appear
> when you press Tab. Press Tab twice to list all candidates when more than one
> matches.

Press **Ctrl-C** (or Ctrl-D at an empty prompt) to exit.

## Project structure

```
src/
  main.ts          Entry point: loads env, spawns the MCP server, runs the REPL
  mcpClient.ts     MCPClient — wraps an MCP client session over stdio
  mcpServer.ts     DocumentMCP — the bundled MCP server (in-memory documents)
  core/
    claude.ts      Claude — thin wrapper over the Anthropic SDK
    chat.ts        Chat — the agent loop (call model, run tools, repeat)
    cliChat.ts     CliChat — @document / /command query preprocessing
    cli.ts         CliApp — the readline REPL with Tab completion
    tools.ts       ToolManager — aggregates and executes MCP tools
tests/             Vitest unit tests
dist/              Compiled output (generated by `npm run build`)
```

## Development

### Adding new documents

Edit the `docs` record in `src/mcpServer.ts` to add or change documents. Each
entry maps a document ID to its text content.

### Connecting additional MCP servers

Pass paths to additional MCP server scripts as command-line arguments; each is
launched as a stdio subprocess with `node` and its tools become available to the
chat:

```bash
npm start -- ./path/to/another-server.js
# or, in dev:
npm run dev -- ./path/to/another-server.js
```

### How it fits together

`main.ts` loads `.env`, validates `CLAUDE_MODEL` and `ANTHROPIC_API_KEY`, and
launches the bundled DocumentMCP server as a stdio subprocess. `CliApp` runs the
REPL; each input is preprocessed by `CliChat` (resolving `@document` mentions and
`/command` prompts) and handed to `Chat`, which drives the Anthropic API and
executes any requested MCP tools via `ToolManager` until the model produces a
final answer.

### Tooling

- **Type checking:** strict TypeScript (`npm run typecheck`)
- **Linting:** ESLint with `typescript-eslint` (`npm run lint`)
- **Formatting:** Prettier (`npm run format`)
- **Tests:** Vitest (`npm test`)
- **Circular dependencies:** `madge` (`npm run check:circular`)
